# Yan Linyi的个人主页

这是一个使用HTML、CSS和JavaScript构建的个人主页，包含首页、方法论、文章、关于我、简历下载和联系我等页面。

## 特点

- 响应式设计，适配移动端和桌面端
- 暗色/亮色主题切换
- 粒子背景效果
- 基于GMT+8时间的自动问候

## 页面结构

- `index.html` - 首页
- `playbook.html` - 方法论页面
- `writing.html` - 文章页面
- `about.html` - 关于我页面
- `cv.html` - 简历下载页面
- `contact.html` - 联系我页面

## 技术栈

- HTML5
- CSS3
- JavaScript
- particles.js - 粒子效果
- 自定义主题切换

## 部署

该网站通过Cloudflare Pages部署。
